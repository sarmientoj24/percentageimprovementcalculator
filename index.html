How it works?:

This calculator computes the ratio of change and improvement between two values. Includes time, performance, and speed increase or decrease.

Example:

The old time of execution of a program was 15306 ms and the new time is 799 ms

Time : (old-new)/old x 100%

= (15306-799)/15306 x 100% = 94.7% reduction

Performance : (old-new)/new x 100%

= (15306-799)/799 x 100% = 1816% increase

Speed : old/new

= 15306/799 = 19.16x faster
